Configurá Firebase para sincronizar datos entre dispositivos automáticamente.

⚙️ No configurado
📖 Ver instructivo
  1. Andá a console.firebase.google.com
  2. Creá un proyecto (o usá uno existente)
  3. Authentication → Sign-in method → Habilitá Google
  4. Firestore Database → Creá base de datos → Modo prueba
  5. Project settings → General → Tus apps → Web
  6. Copiá el objeto firebaseConfig y reemplazalo en el archivo HTML (buscá "firebaseConfig" en el código)
  7. ¡Listo! La app sincronizará automáticamente
